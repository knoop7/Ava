# 系统要求

本页面列出运行 Ava 所需的系统要求。

## 为什么选择 Ava？

与 Fully Kiosk Browser 等方案相比，Ava 有以下优势：

| 对比项 | Ava | Fully Kiosk |
|--------|-----|-------------|
| 内存占用 | 100-150 MB | 200-300 MB |
| 显示方式 | 悬浮窗（效果更好） | 内嵌浏览器 |
| 语音助手 | 原生支持 | 需额外配置 |
| 价格 | 免费开源 | 付费 |

更多信息：https://www.fully-kiosk.com/

---

## Android 设备要求

| 要求 | 最低版本 | 推荐版本 |
|------|---------|----------|
| Android 版本 | 7.0 (API 24) | 10.0 或更高 |
| CPU 架构 | armeabi-v7a | arm64-v8a |
| 内存 | 1GB | 2GB 或更高 |
| 存储空间 | 100MB | 200MB |

### A64 设备特别优化

Ava 对 A64 芯片设备（如部分全志平板、智能屏等）进行了特别优化：

- 自动检测 A64 设备类型
- 针对低端硬件优化性能
- 支持 Android 7.0-8.0 的老旧设备

## 网络要求

| 要求 | 说明 |
|------|------|
| WiFi | 设备需要连接到与 Home Assistant 相同的局域网 |
| 端口 | 6053 (ESPHome 默认端口) 需要可访问 |
| 网络稳定性 | 建议使用稳定的 WiFi 连接 |

---

## Home Assistant 要求

| 组件 | 说明 |
|------|------|
| Home Assistant | 需要已安装并运行 |
| ESPHome 集成 | 需要在 HA 中安装 ESPHome 集成 |
| 语音助手 | 需要配置语音助手管道 (可选) |

### 推荐的语音助手组件

| 组件 | 用途 | 说明 |
|------|------|------|
| Whisper | 语音转文字 | 本地运行，隐私保护 |
| Piper | 文字转语音 | 本地运行，速度快 |
| Home Assistant 对话代理 | 意图处理 | 内置，无需额外配置 |

---

## 权限要求

### 必需权限

| 权限 | 用途 |
|------|------|
| RECORD_AUDIO | 麦克风录音，用于语音输入 |
| SYSTEM_ALERT_WINDOW | 悬浮窗，用于显示屏保、通知等 |
| FOREGROUND_SERVICE | 前台服务，保持服务运行 |
| INTERNET | 网络访问，与 HA 通信 |
| WAKE_LOCK | 唤醒锁，保持设备唤醒 |

### 可选权限

| 权限 | 用途 |
|------|------|
| CAMERA | 摄像头功能 |
| BLUETOOTH | 蓝牙在场检测 |
| ACCESS_FINE_LOCATION | 蓝牙扫描需要 |
| WRITE_SETTINGS | 调节屏幕亮度 |

## 设备能力检测

Ava 会自动检测设备能力，根据硬件情况调整功能：

| 检测项 | 用途 |
|--------|------|
| 摄像头 | 自动检测前置/后置摄像头 |
| 光照传感器 | 光照感应关屏功能 |
| 接近传感器 | 接近感应唤醒功能 |
| 温度/湿度/气压传感器 | 环境传感器数据 |
| A64 设备类型 | 低端设备优化 |

## 推荐设备

### 适合作为中控屏的设备

| 设备类型 | 优点 | 缺点 |
|---------|------|------|
| 旧手机 | 便宜，容易获得 | 屏幕较小 |
| 平板电脑 | 屏幕大，显示效果好 | 需要固定放置 |
| 触屏音箱 | 音质好，麦克风质量高 | 专用设备 |
| 带屏幕的 Android 设备 | 专为此设计 | 可能较贵 |

### 设备选择建议

1. **屏幕大小**: 7-10 英寸最适合作为中控屏
2. **麦克风**: 确保麦克风质量良好，用于语音识别
3. **扬声器**: 确保扬声器音质可接受
4. **触屏音箱**: 支持触屏音箱设备，如小米触屏音箱等

### 噪音消除技术

Ava 内置噪音消除技术，可以在喧杂环境中提高语音识别准确率。

## 已知兼容性问题

| 问题 | 影响设备 | 解决方案 |
|------|---------|----------|
| WebView 渲染问题 | 部分老旧设备 | 切换到软件渲染模式 |
| 唤醒词检测不准 | 麦克风质量差的设备 | 靠近设备说话 |
| 悬浮窗权限 | 部分国产 ROM | 手动在设置中授权 |
