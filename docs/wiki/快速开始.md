# 快速开始

本指南帮助你快速安装和配置 Ava。

---

## 系统要求

| 要求 | 说明 |
|------|------|
| Android 版本 | 7.0 (API 24) 或更高 |
| 架构 | arm64-v8a 或 armeabi-v7a |
| Home Assistant | 需要已安装并运行 |
| ESPHome 集成 | 需要在 HA 中安装 |
| 网络 | 设备和 HA 需在同一局域网 |

---

## 第一步：安装 Ava

### 下载 APK

从 GitHub Releases 下载最新版本：
- https://github.com/knoop7/Ava/releases

### 安装

1. 将 APK 文件传输到你的 Android 设备
2. 点击 APK 文件开始安装
3. 如果提示"未知来源"，请允许安装

### 授予权限

首次启动时，Ava 会请求以下权限：

| 权限 | 用途 | 必需 |
|------|------|------|
| 麦克风 | 语音输入 | ✅ 是 |
| 悬浮窗 | 显示屏保、通知等 | ✅ 是 |
| 前台服务 | 保持服务运行 | ✅ 是 |
| 摄像头 | 拍照和视频 | ❌ 可选 |
| 蓝牙 | 在场检测 | ❌ 可选 |
| 位置 | 蓝牙扫描需要 | ❌ 可选 |

---

## 第二步：配置 Home Assistant

### 安装 ESPHome 集成

1. 打开 Home Assistant
2. 进入 **设置** → **设备与服务** → **集成**
3. 点击右下角 **+ 添加集成**
4. 搜索 **ESPHome**
5. 点击安装

### 发现 Ava 设备

1. 启动 Ava 应用
2. 进入 **设置** → **语音卫星**
3. 点击 **启动服务**
4. 回到 Home Assistant
5. ESPHome 集成会自动发现 Ava 设备
6. 点击 **配置** 完成添加

### 手动添加（如果自动发现失败）

1. 在 ESPHome 集成中点击 **配置**
2. 输入 Ava 设备的 IP 地址
3. 端口默认为 6053
4. 点击提交

---

## 第三步：配置语音助手

### 在 Home Assistant 中

1. 进入 **设置** → **语音助手**
2. 确保已配置语音助手管道
3. 推荐配置：
   - 语音转文字：Whisper
   - 文字转语音：Piper
   - 对话代理：Home Assistant 内置

### 在 Ava 中

1. 进入 **设置** → **语音卫星**
2. 确认状态显示"已连接"
3. 选择你喜欢的唤醒词

---

## 第四步：测试

### 语音测试

1. 说出唤醒词（默认 "Hey Jarvis"）
2. 听到提示音后说 "What time is it"
3. Ava 应该会回复当前时间

### 控制测试

1. 说出唤醒词
2. 说 "Turn on the living room light"（假设你有这个设备）
3. 灯应该会打开

---

## 常见问题

### 设备无法被发现？

1. 确保 Ava 和 Home Assistant 在同一网络
2. 检查防火墙是否阻止了 6053 端口
3. 尝试重启 Ava 服务
4. 尝试手动添加设备

### 连接后断开？

1. 检查网络稳定性
2. 确保 Ava 没有被系统杀死
3. 在设置中关闭电池优化

### 语音识别不工作？

1. 检查 Home Assistant 的语音助手配置
2. 确保 Whisper 等组件正常运行
3. 检查网络延迟
