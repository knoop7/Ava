# 屏保系统

屏保系统让你的设备在空闲时显示漂亮的内容，同时保护屏幕。

---

## 概述

当设备空闲一段时间后，Ava 会自动显示屏保。触摸屏幕即可退出屏保。

**屏保类型：**
- 小米画报（推荐）
- 网页屏保
- 图片屏保

---

## 小米画报

小米画报是 Ava 内置的精美屏保，模仿小米手机的画报风格。

### 显示内容

```
┌─────────────────────────────────────┐
│                                     │
│           12:34                     │
│                                     │
│     2026年1月9日 星期四              │
│                                     │
│                                     │
│     春眠不觉晓，处处闻啼鸟           │
│              ——唐·孟浩然             │
│                                     │
└─────────────────────────────────────┘
```

### 功能特点

| 功能 | 中文环境 | 英文环境 |
|------|---------|---------|
| 时间 | 24小时制 | 24小时制 |
| 日期 | 2026年1月9日 星期四 | Thursday, January 9, 2026 |
| 古诗词 | 显示（每次刷新更换） | 隐藏 |

### 如何开启

1. 进入 **设置** → **屏保**
2. 打开 **屏保开关**
3. 打开 **小米画报** 开关
4. 设置 **超时时间**（多久不操作后显示屏保）

### 古诗词来源

古诗词来自「今日诗词」API，每次刷新随机显示一句古诗词。

## 网页屏保

网页屏保可以显示任何网页内容，比如天气、日历、仪表盘等。

### 使用场景

- 显示 Home Assistant 仪表盘
- 显示天气网站
- 显示自定义网页

### 如何设置

1. 进入 **设置** → **屏保**
2. 打开 **屏保开关**
3. 关闭 **小米画报** 开关
4. 在 **屏保网址** 中输入网页地址
5. 设置 **超时时间**

### 推荐网页

| 网页 | 说明 |
|------|------|
| file:///android_asset/xiaomi_wallpaper.html | 内置小米画报 |
| http://your-ha:8123/lovelace/screensaver | HA 屏保仪表盘 |
| https://www.bing.com | 必应首页 |

---

## 图片屏保

图片屏保可以显示网络图片，支持 MJPEG 视频流。

### 支持的格式

- JPEG/JPG
- PNG
- GIF
- MJPEG 视频流

### 如何设置

通过 Home Assistant 服务设置图片 URL：

```yaml
service: esphome.your_device_name_screensaver_image
data:
  url: "http://example.com/image.jpg"
```

## 屏保控制

### 超时时间

设置多久不操作后显示屏保。

| 选项 | 说明 |
|------|------|
| 15秒 | 快速进入屏保 |
| 30秒 | 默认值 |
| 60秒 | 较长等待 |
| 120秒 | 很长等待 |
| 300秒 | 5分钟 |

### 触摸唤醒

触摸屏幕任意位置即可退出屏保。

### 光照感应关屏

当环境光线很暗时（如晚上关灯），自动关闭屏幕以省电。

**如何开启：**
1. 进入 **设置** → **屏保**
2. 打开 **光照感应关屏** 开关

**工作原理：**
- 光照低于 2 lux 时关闭屏幕
- 光照恢复后自动开启屏幕

### 接近感应唤醒

当有人靠近设备时，自动唤醒屏幕。

**如何开启：**
1. 进入 **设置** → **屏保**
2. 打开 **接近感应唤醒** 开关

**工作原理：**
- 使用设备的接近传感器
- 检测到物体靠近时唤醒屏幕

## 设置汇总

| 设置项 | 位置 | 说明 | 默认值 |
|--------|------|------|--------|
| 屏保开关 | 屏保 | 启用/禁用屏保 | 关闭 |
| 小米画报 | 屏保 | 使用内置小米画报 | 关闭 |
| 屏保网址 | 屏保 | 自定义网页屏保地址 | 空 |
| 超时时间 | 屏保 | 多久后显示屏保 | 30秒 |
| 光照感应关屏 | 屏保 | 暗光自动关屏 | 关闭 |
| 接近感应唤醒 | 屏保 | 靠近自动唤醒 | 关闭 |

## Home Assistant 控制

屏保通过开关实体和文本实体控制。

### 显示/隐藏屏保

```yaml
# 显示屏保
service: switch.turn_on
target:
  entity_id: switch.your_device_name_screensaver

# 隐藏屏保
service: switch.turn_off
target:
  entity_id: switch.your_device_name_screensaver
```

### 设置屏保图片 URL

```yaml
service: text.set_value
target:
  entity_id: text.your_device_name_screensaver_image_url
data:
  value: "http://example.com/image.jpg"
```

## 常见问题

### 屏保不显示？

1. 检查屏保开关是否打开
2. 检查悬浮窗权限是否已授予
3. 检查超时时间设置
4. 如果使用网页屏保，检查网址是否正确

### 屏保显示空白？

1. 检查网络连接
2. 检查网址是否可访问
3. 尝试使用内置小米画报

### 光照感应不工作？

1. 检查设备是否有光照传感器
2. 确保传感器没有被遮挡
3. 检查权限设置
